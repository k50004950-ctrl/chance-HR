## 🗺️ 외부 API 및 서비스

### 1. Kakao Maps API

**현재 사용:**
- 지도 표시
- 주소 → 좌표 변환
- 장소 검색

**라이선스:**
- ✅ **무료 Tier**: 월 300,000건까지 무료
- ⚠️ **상업용**: 트래픽 많으면 유료 전환 필요
- 📄 **약관**: https://apis.map.kakao.com/terms/

**비용 (유료 전환 시):**
- 월 50만건: 약 10만원
- 월 100만건: 약 18만원
- 소규모 사업장: 무료로 충분할 가능성 높음

**API 키:**
- 현재 코드에 하드코딩됨: `f08c77bfb5eb0bcf42a30ed4982c94f2`
- ⚠️ **보안 주의**: `.env`로 이동 권장

**대안:**
- Google Maps API (유사한 가격 정책)
- Naver Maps API (국내 서비스)
- OpenStreetMap (완전 무료, 정확도는 낮음)

### 2. Daum 우편번호 서비스

**현재 사용:**
- 주소 검색 팝업

**라이선스:**
- ✅ **무료**: 상업용 포함 무료
- 📄 **약관**: https://postcode.map.daum.net/guide

**조건:**
- 출처 표시 필요 없음
- 상업용 사용 제한 없음
- API 호출 제한 없음

**✅ 결론:** 완전 무료, 상업용 사용 문제 없음

### 3. Nominatim (OpenStreetMap)

**현재 사용:**
- Kakao Maps 실패 시 Fallback
- 주소 → 좌표 변환

**라이선스:**
- ✅ **무료**: ODbL 라이선스
- ⚠️ **Usage Policy**: 초당 1회 요청 제한

**상업용 사용:**
- ✅ 가능 (라이선스 명시 필요)
- 라이선스 텍스트: "© OpenStreetMap contributors"

**추천:**
- 트래픽 많으면 자체 Nominatim 서버 구축
- 또는 상업용 Geocoding API 사용
