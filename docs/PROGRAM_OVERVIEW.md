## ⭐ 주요 기능

### 📍 GPS 출퇴근 관리
- 위치 기반 출퇴근 인증으로 정확한 근태 관리
- 지각, 조퇴, 결근 자동 체크
- 실시간 출퇴근 알림

### 💰 자동 급여 계산
- 월급/시급 자동 계산
- 주휴수당, 연장수당, 야간수당 포함
- 4대보험 및 소득세 자동 산출

### 📄 급여명세서 발급
- 전자 급여명세서 자동 생성 및 배포
- 언제 어디서나 모바일로 확인 가능
- 과거 명세서 보관 및 재발급

- 직원 정보 통합 관리
- 계약서류 보관
- 입퇴사 처리
- 근무 스케줄 설정

### 📊 근태 현황 분석
- 캘린더 뷰로 한눈에 보는 근태 현황
- 지각/결근/연차 통계
- 리스크 직원 자동 알림

### 🧮 퇴직금 계산
- 근속기간 자동 계산
- 퇴직금 시뮬레이션

### 💬 커뮤니티
- 사업주/직원 간 소통 공간
- 공지사항 전달
- 실시간 알림 기능

### 📱 모바일 최적화

## 💡 사용 사례

### Case 1: 음식점 사업주
- **Before**: 수기로 출퇴근 체크, 엑셀로 급여 계산 → 실수 빈번
- **After**: GPS로 자동 출퇴근, 급여 자동 계산 → 월 10시간 절약

### Case 2: 소규모 제조업
- **Before**: 복잡한 교대 근무, 수당 계산 어려움
- **After**: 스케줄 관리 자동화, 정확한 수당 계산 → 직원 만족도 UP

### Case 3: 복수 매장 운영
- **Before**: 매장별 근태 관리 힘듦, 통합 관리 어려움
- **After**: 모든 매장 한눈에 파악, 실시간 모니터링 → 효율적 운영
