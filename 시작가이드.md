# 출퇴근 관리 시스템 - 시작 가이드

## ✅ 현재 실행 상태

### 백엔드 서버
- **상태**: ✅ 실행 중
- **주소**: http://localhost:5000
- **터미널**: Terminal 3

### 프론트엔드 서버
- **상태**: ✅ 실행 중
- **주소**: http://localhost:3000
- **터미널**: Terminal 5

## 🚀 빠른 시작

### 1. 웹 브라우저에서 접속
```
http://localhost:3000
```

### 2. 관리자 계정으로 로그인
- **사용자명**: `admin`
- **비밀번호**: `admin123`

## 📋 주요 기능 사용 방법

### 관리자 (Admin)
1. **사업장 등록**
   - 사업장 관리 탭 선택
   - "+ 사업장 등록" 버튼 클릭
   - 사업장명, 주소, 위도, 경도, 반경 입력
   - 저장

2. **사업주 계정 생성**
   - 사업주 관리 탭 선택
   - "+ 사업주 등록" 버튼 클릭
   - 사용자명, 비밀번호, 이름 등 입력
   - 등록

### 사업주 (Owner)
1. **로그인 후 사업장 선택**
   - 자신의 사업장이 자동으로 표시됨

2. **직원 등록**
   - 직원 관리 탭 선택
   - "+ 직원 등록" 버튼 클릭
   - 기본 정보 입력:
     - 사용자명, 비밀번호, 이름
     - 전화번호, 이메일
     - 입사일, 직책, 부서
   - 급여 정보 입력:
     - 급여 유형 선택 (시급/월급/연봉)
     - 금액 입력
     - 시급인 경우 주휴수당 적용 체크
   - 근로계약서 파일 업로드 (선택사항)
   - 저장

3. **출퇴근 기록 확인**
   - 출퇴근 기록 탭 선택
   - 월 선택하여 조회
   - 직원별 출퇴근 시간, 근무시간 확인

4. **급여 계산**
   - 급여 계산 탭 선택
   - 월 선택하여 조회
   - 직원별 급여 자동 계산 결과 확인
   - 총 지급 급여 확인

### 직원 (Employee)
1. **로그인**
   - 사업주가 등록한 사용자명/비밀번호로 로그인

2. **출근하기**
   - "출근하기" 버튼 클릭
   - 브라우저에서 위치 권한 허용
   - 사업장 범위 내에 있으면 출근 기록됨

3. **퇴근하기**
   - "퇴근하기" 버튼 클릭
   - 위치 확인 후 퇴근 기록
   - 자동으로 근무시간 계산됨

4. **출퇴근 기록 확인**
   - 월 선택하여 자신의 출퇴근 기록 조회
   - 날짜별 출근/퇴근 시간, 근무시간 확인

5. **급여 확인**
   - 이번 달 급여 자동 계산 결과 확인
   - 근무일수, 총 근무시간, 예상 급여 확인

## 🌍 위치 기반 출퇴근

### 위치 설정 방법
1. **Google Maps에서 위도/경도 찾기**
   - Google Maps 열기
   - 사업장 위치 클릭
   - 하단에 표시되는 좌표 확인
   - 예: `37.5665, 126.9780` (서울시청)

2. **반경 설정**
   - 기본값: 100미터
   - 사업장 크기에 맞게 조정 가능
   - 예: 작은 사무실 50m, 큰 공장 200m

### 위치 권한 허용
- 브라우저에서 위치 권한 요청 시 "허용" 클릭
- Chrome: 주소창 왼쪽 자물쇠 아이콘 → 위치 → 허용

## 💰 급여 계산 방식

### 시급
```
기본급 = 총 근무시간 × 시급
주휴수당 = (주당 근무시간 × 시급 × 0.2) × 주 수
※ 주 15시간 이상 근무 시 주휴수당 적용
```

**예시:**
- 시급: 10,000원
- 한 달 근무시간: 160시간
- 기본급: 1,600,000원
- 주휴수당 (주 40시간 근무): 320,000원
- **총 급여: 1,920,000원**

### 월급
```
총 급여 = 월급 × 개월 수
```

### 연봉
```
월 급여 = 연봉 ÷ 12
```

## 📱 모바일에서 사용

1. 컴퓨터와 같은 Wi-Fi 네트워크에 연결
2. 컴퓨터의 IP 주소 확인:
   ```powershell
   ipconfig
   ```
3. 모바일 브라우저에서 접속:
   ```
   http://[컴퓨터IP]:3000
   ```
   예: `http://192.168.0.10:3000`

## ⚠️ 주의사항

1. **위치 권한**
   - 출퇴근 체크 시 위치 권한 필수
   - HTTPS 환경 또는 localhost에서만 작동

2. **보안**
   - 관리자 비밀번호 반드시 변경
   - 프로덕션 환경에서는 JWT_SECRET 변경

3. **데이터 백업**
   - 데이터베이스 파일: `backend/database.db`
   - 업로드된 파일: `backend/uploads/`
   - 정기적으로 백업 권장

## 🐛 문제 해결

### 로그인이 안 돼요
- 사용자명/비밀번호 확인
- 백엔드 서버 실행 상태 확인 (http://localhost:5000)

### 출퇴근 버튼이 작동하지 않아요
- 위치 권한 허용 확인
- 사업장 범위 내에 있는지 확인
- 브라우저 콘솔(F12) 에러 메시지 확인

### 급여가 계산되지 않아요
- 급여 정보가 등록되어 있는지 확인
- 출퇴근 기록이 있는지 확인
- 해당 월의 데이터가 있는지 확인

## 📞 지원

문제가 발생하면:
1. 브라우저 콘솔(F12) 확인
2. 백엔드 터미널 로그 확인
3. 프론트엔드 터미널 로그 확인
